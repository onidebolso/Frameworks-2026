# Mecânicas Básicas de Death Stranding

Dominar as mecânicas é essencial para sobreviver no mundo de Death Stranding. Aqui vai um guia das principais.

## Movimento e Terreno
- **Caminhada e Corrida**: Use analog para mover Sam. Pressione para correr, mas cuidado com a stamina.
- **Escalada**: Segure para escalar superfícies. Use ferramentas como ganchos para terrenos difíceis.
- **Natação**: Sam pode nadar, mas evite águas profundas com BTs.

## Entregas
- **Pacotes**: Carregue ordens e entregue-as. Use o mapa para planejar rotas.
- **Veículos**: Desbloqueie bikes, caminhões e motos para transportes maiores.
- **Construção**: Construa estradas e pontes para rotas seguras. Use materiais coletados.

## Combate e Sobrevivência
- **Armas**: Pistolas, rifles e granadas. Foque em defesa, não ataque.
- **BTs**: Evite ou use ferramentas como o Odradek para detectar.
- **Mules**: Ladrões que roubam entregas. Use armadilhas ou lute.

## Dicas Avançadas
- Mantenha o equilíbrio chiral alto para evitar quedas.
- Conecte-se com outros jogadores via chiral network para itens compartilhados.
- Explore para encontrar recursos raros.
